✨mp4cut.js✨

JS & browser based playback & clipping of .mp4 files

Rewrites header and only (losslessly) sends desired A/V packets for the desired start/end range

Repo: https://github.com/traceypooh/mp4cut.js
Online demo: https://traceypooh.github.io/mp4cut.js
Uses mp4/moov parsing JS from mp4box.js

Uses the Media Source Extensions (MSE) API
https://hacks.mozilla.org/2015/07/streaming-media-on-demand-with-media-source-extensions/
to inject video to this <video> tag
media files served locally need to be served using something like
ruby -run -e httpd . -p 8888
for http RANGE requests, and the server need to support CORS options)
